# :rocket: Tecnologias

- React
- Styled Components
- Context API
- Axios
- react-tabs
- API REST do GitHub

<br/>

# :information_source: Como usar

```bash
# Clone este repositório:
$ git clone https://github.com/JonataRamalho/github-api.git

# Vá para o repositório:
$ cd github-api

# Instale as dependências:
$ yarn install

# Inicie o React:
$ yarn start
```
